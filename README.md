## ⚒️ Features

- **This app will let you depict a stack**

- **You can choose any type - string, int, float, double...**

- **There is an option for User Mode Insertion or Random Insertion**


## ❗Usage

**1) Push**

**2) Pop**

**3) Top**

**4) Print**
